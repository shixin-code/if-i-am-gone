# 受益人加密文件下载与解压说明

这份说明给收到邮件的受益人使用。你会先收到一封预提醒邮件，随后可能收到“解压密码”邮件和“加密文件下载链接”邮件。只有同时拿到密码和下载链接，才能打开文件。

## 你会收到什么

通常会按顺序收到三类邮件：

1. 预提醒邮件：说明系统即将发送重要信息，并告知预计日期。
2. 解压密码邮件：包含用于打开加密 ZIP 文件的密码。
3. 下载链接邮件：包含加密 ZIP 文件的下载链接、有效期和最大下载次数。

如果设置人后来通过 Telegram 完成确认，后续邮件可能会停止发送。

## 下载文件

收到下载链接邮件后：

1. 点击邮件中的下载链接。
2. 将文件保存到本地电脑，不要只在浏览器里预览。
3. 如果浏览器提示文件可能有风险，请确认来源是你认识的人预设的邮件流程后再继续。
4. 下载后不要随意转发链接。链接有有效期和下载次数限制。

## 推荐解压工具

加密文件是 AES-256 加密 ZIP。推荐使用：

- Windows：7-Zip 或 WinRAR。
- macOS：Keka 或 The Unarchiver。
- Linux：7-Zip / p7zip。

Windows 自带解压工具通常不支持此类 AES 加密 ZIP。如果双击打不开、提示密码错误或文件无效，请先换 7-Zip 或 WinRAR 再试。

## Windows 解压步骤

以 7-Zip 为例：

1. 安装 7-Zip。
2. 右键下载到的 `.zip` 文件。
3. 选择 `7-Zip` -> `Extract Here` 或 `解压到当前文件夹`。
4. 在弹出的密码框中输入“解压密码”邮件里的密码。
5. 解压完成后，在当前目录查看解压出的文件。

如果使用 WinRAR：

1. 右键 `.zip` 文件。
2. 选择 `解压到...`。
3. 输入密码。
4. 等待解压完成。

## macOS 解压步骤

以 Keka 为例：

1. 安装 Keka。
2. 用 Keka 打开下载到的 `.zip` 文件，或把文件拖到 Keka。
3. 输入“解压密码”邮件里的密码。
4. 选择保存位置并等待解压完成。

如果系统自带归档工具提示无法解压，请改用 Keka。

## 常见问题

### 提示密码错误

请确认：

- 复制的是“解压密码”邮件里的完整密码。
- 没有多复制空格、换行或中文冒号。
- 使用的是 7-Zip、WinRAR 或 Keka，而不是系统自带解压工具。

### 链接打不开

可能原因：

- 链接已经过期。
- 下载次数已经达到上限。
- 网络或浏览器拦截。
- 服务器暂时不可用。

可以稍后重试。如果仍无法下载，请联系设置人预先指定的备用联系人或管理员。

### 文件下载了但打不开

请先确认文件下载完整。不要在下载未完成时解压。如果文件大小明显异常，重新下载一次。若仍失败，换用推荐解压工具。

## 安全提醒

- 不要把密码和下载链接转发到公开群聊。
- 不要把解压后的文件上传到不可信网盘或公共电脑。
- 如果内容涉及身份证件、账号、财产、家庭信息，请优先离线保存并谨慎处理。
- 如果你不确定邮件是否真实，请先通过电话或其他可信渠道联系设置人或家人确认。
